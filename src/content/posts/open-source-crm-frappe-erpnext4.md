## 6. Hướng dẫn cài đặt chi tiết (Self-host)
Để tự host [Tên Repo] trên máy chủ Ubuntu, bạn có thể làm theo các bước sau:

### Yêu cầu hệ thống
- Ubuntu 20.04 LTS trở lên
- Docker (phiên bản mới nhất)
- [Các yêu cầu khác nếu có]

### Các bước cài đặt
1. **Cài đặt Docker**
   ```bash
   sudo apt-get update
   sudo apt-get install docker.io
   sudo systemctl enable --now docker
   ```

2. **Clone repository**
   ```bash
   git clone [URL repository]
   cd [tên thư mục repository]
   ```

3. **Chạy ứng dụng với Docker**
   ```bash
   docker-compose up -d
   ```

4. **Truy cập ứng dụng**
   Mở trình duyệt và truy cập `http://localhost:[port]` để bắt đầu sử dụng.

Với hướng dẫn trên, bạn đã có thể triển khai [Tên Repo] một cách dễ dàng. Chúc bạn thành công!
